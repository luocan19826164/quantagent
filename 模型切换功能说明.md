# 模型动态切换功能

现在前端可以直接在界面上切换模型了！

## 功能位置

在**对话区右上角**有一个模型选择下拉框，可以切换不同的模型。

## 当前支持的模型

- **GPT-4o Mini** (默认，推荐)
- **GPT-4o**
- **GPT-4 Turbo**

## 使用方法

1. 在对话区右上角的下拉框选择模型
2. 确认切换（会话历史和上下文保持不变）
3. 继续对话使用新模型

## 工作流程

```
用户选择模型 
  ↓
确认切换 
  ↓
后端切换LLM实例（保持state和memory不变）
  ↓
新的对话使用新模型
```

## 注意事项

⚠️ **目前只配置了OpenAI**，暂时不支持DeepSeek（需要配置API Key）

配置DeepSeek需要：
1. 在 `.env` 文件添加 `DEEPSEEK_API_KEY=xxx`
2. 重启应用
3. 然后在代码中添加DeepSeek选项

## 技术实现

- 前端：下拉框 + JavaScript事件监听
- 后端：切换LLM实例，保持上下文
- 无状态丢失：切换模型不影响对话历史和状态

## 后续扩展

等申请了DeepSeek API Key后，可以：
1. 在 `.env` 添加 `DEEPSEEK_API_KEY`
2. 在 `app.py` 的 `SUPPORTED_MODELS` 添加DeepSeek配置
3. 在前端HTML添加DeepSeek选项

无需重启应用，直接刷新页面即可！
